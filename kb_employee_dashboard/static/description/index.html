Employee Dashboard

Employee Dashboard Module to enable employees to easily access all data.


    To allow the user to access the dashboard, you can choose Employee Dashboard from menu.
    Through the Dashboard,
    if you want to view your leaves, you can click on "Time Off."
    If you want to see your contracts, you can click on "Contract."
    If you want to view salary details, you can click on "Payslip."
    And if you want to check attendance, you can click on the "More" button in the attendance table.
    In the settings, select "Employee readonly" from the Employee Dashboard category.
