In any industry today inventory optimization is such a vital function. It’s hard to track inventory movements, especially when you have a large products catalog, multiple warehouses or multi company environment.

It’s necessary to know inventory movements each day with all possible movements for each product for each warehouse or company and it’s very difficult to manage. So we introduced a solution for that and built an Inventory Ledger Report where users can track each day's transactions or summary of all dates together with real calculated opening and closing. This app helps you to track the inventory for specific periods or up to a certain date.


Key Features


  • Detailed and Summary reports available
    •
  • Report can be downloaded by company or warehouse, for both summary report and detailed reports are available
    •
  • Date wise calculated opening stock and closing stock
    •
  • Date wise detailed data for each movement.
    •
  • Capture all possible inventory movements and separately visible all IN transactions and OUT transactions in report. (Date wise / Summary).
    •
  • Report can be visible in Odoo list view, where user can perform grouping, advance searching and filtration easily.
    •
  • Quick filters for company, warehouse, product category and products to generate reports for specific dataset.
    •
  • Users can perform inventory comparison to compare calculated inventory v/s theoretical inventory available in the system.
