Include Weekend in Holidays

This Module add an option to Leave Type for include the Weekend in Time-Off


    From odoo menu go to "Time Off"



    Click on "configuration" and select "Time Off Types"



    Select any leave you want to include the weekend



    Uncheck the box to activate the option then click "save button"
