Discount On Sale Order

Increase sales by offering discounts in Odoo!

Discount on Sale Order adds a new type of discount in Odoo. It allows you to add a discount to a sales order which is not present by default. We added a ’Global Discount Category’ offering fixed or percentage types discounts that you can apply to the entire sales order.

Why Do You Need a Discount On the Sale Order Module?

You need Odoo Discount on Sale Order module because Odoo only authorizes percent discounts in sales order lines. The module adds more flexible options to offer various discounts in Odoo. It allows you to add discounts per each order line or globally.

The Discount On Sale Order permits you to manage discount types (Fixed and Percentage). And it gives the flexibility to choose the kind of discount categories (Order line or global).

Further, the sales order discount module provides multiple options to apply discounts in Odoo. In addition, you can set different types of discounts in an Odoo sales order.

Offer Fixed/Flat Discount Type on Sales Order Lines in Odoo

According to the default Odoo functionality, Odoo only allows you to add percentage discounts in sales order lines.

But, with the Discount on sales order module, you can add fixed/flat discount type to the sales order lines. Also, you can set the new product price in the price list, and the module will show the difference between the latest (new) and original sale price as a fixed discount on the sales order line.

The discount amount can be taxed or untaxed.

Global Discount Category For Entire Sale Order

Note:By default, Odoo does not allow adding a global discount means a discount on the entire amount or sales order.

However, the Discount on Sale Order module permits you to apply a global discount on the entire amount or sales order. The sale order discount module further authorizes adding fixed or percentage discount types on the complete sales order. The admin can set the settings to give discounts on taxed or untaxed sales order amounts.

Apply Different Types of Discounts on the Order Lines

  • The module permits you to apply a fixed or flat discount on sales order lines.
  • You can also add a percentage discount on the sales order lines.

Set Global Discount on the Complete Sales Order in Odoo

  • Discount on sales order module allows you to apply a fixed global discount on the complete sales order.
  • Or you can apply a global discount on the entire sales order.

Configure Global Discount from the Odoo Backend

  • You can enable or disable the global discount settings from the Odoo backend.
  • Additionally, you can manage a separate account for a global discount on sales orders from the settings.

Apply Discount on Various Amounts

  • You can apply a discount on taxed or untaxed amounts according to the set configuration from the Odoo backend.
  • Moreover, the applied discount details are also carried forward to the sales order reports.
