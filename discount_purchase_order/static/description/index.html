Odoo Discount On Purchase Order

By default, you cannot add any discount in a purchase order. So, the module adds new discount types to Odoo purchase orders. Now you can add a FIXED or PERCENT Type purchase orderline Discount. Also, we have added a GLOBAL DISCOUNT CATEGORY which can be fixed or percentage type and apply on the entire purchase order.

Note: The module works in conjunction with Odoo Discounts on Invoices and Bills. You need to install the module Odoo Discounts on Invoices and Bills for the module to work properly.

While Odoo Discounts on Invoices and Bills module gives discount options while creating invoices and bills, this module provides options to set various discounts on Purchase Orders in Odoo.

Manage Discounts On Purchase Order In Odoo

  • Set different types of discounts are different levels in a Odoo purchase order.

Adding Discount Types - Fixed And percent Discounts On Purchase OrderLines In Odoo

Default Odoo Flow

  • There is no discount field in the Orderlines of purchase orders.

With Our Module

  • You can now also add a Fixed or Percent Discount type in the order lines of the purchase order.
  • The discount can be applied on taxed or untaxed amounts of the product.

Introducing Global Discount Category For the Entire Purchase Order Amount

Note: By default, you can not add any global discount i.e. discount on the entire purchase order amount.

With our module

  • Global discount is applied on the total purchase order amount.
  • The module allows you to add fixed or percent discount type on the whole purchase order.
  • Again the discount can be given on taxed and untaxed purchase order amount as set in the settings.

Summarized Features List

  • Apply fixed and percent type discount on Odoo Purchase orderlines.
  • Apply Fixed and percent type Global discount on the entire purchase order.
  • Enable/Disable Global Discount From the Configuration Settings.
  • Apply discounts on taxed and untaxed amount (easily set from configuration settings).
  • Import the applied discount to the purchase order reports.
  • Manage Separate Account for Global Discount from the settings.
  • The discounts applied on purchase orders are also carried automatically forward to the generated invoices with the help of base module - Odoo Discounts On Invoices and Bills.
